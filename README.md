# API de demonstração de Estrutura de Dados com Spring Boot
API de demonstração para palestra na Comunidade Devs Java Brasil sobre Descomplicando Estruturas de Dados com Java 25 e Spring Boot. 
Aplicando estruturas de dados estão presentes no coração das APIs modernas, tais como List, Map, Queue e Stack em uma aplicação real construída com Spring Boot e Postman, aplicando boas práticas e recursos do Java 25, as quais são a base que impulsiona a performance, a clareza e a eficiência das nossas aplicações.
